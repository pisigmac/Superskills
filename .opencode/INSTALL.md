> **For OpenCode users:** Add the following entry to the `plugin` array in your `opencode.json` (global or project-level), then restart OpenCode.

```json
{
  "plugin": ["superskills@git+https://github.com/pisigmac/Superskills.git"]
}
```

After restart, verify by asking: `Tell me about your superskills`

For full usage details, see [`docs/README.opencode.md`](../docs/README.opencode.md).
